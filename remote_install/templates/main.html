Welcome

This application supports the remote testing of Fedora's Grapical installation.

This allows you to try a custom installation of Fedora in a remote virtual machine without needing to download ISOs or use your own hardware. Play with the interface, try as many different installations as you would like and if you do hit issues, please report them at the project site.

WARNING

This application is ONLY a demo, it might (and probably will) break. If you do hit a bug, please file an issue at the project site.

Basic Usage Instructions

  • To get started, create a new job using the button in the upper left hand side of the page. VM creation is currently very slow, please wait until the page has reloaded after job creation before clicking anything else.
  • Once the page has reloaded, initialization of the job will take 30 seconds to one minute. The job detail page will change (manual refresh required) when initialization is complete and login instructions will be displayed.
  • Login to the virtual machine by clicking on the 'remote access' link on the page and logging in to the virtual machine's VNC console.
  • Once installation is complete, you will need to reboot the VM from this web UI before logging in to the installed machine.
